# 컴성비 프로젝트

Title: 컴성비프로젝트<br>
컴성비프로젝트는  컴퓨터의  부품의  정보를  찾아보고싶은  사용자들과<br> 
컴퓨터에  대해  잘모르는  사용자들을  위해  만들어진<br>
컴퓨터  부품정보  사이트  입니다.<br>

개발인원 : 3<br>

사용환경 <br>
1.HTML <br>
2.JavaScript <br>
3.spring<br>
4.css<br>
5.msyql<br>

<br>
목표
컴성비 프로젝트는 크게 두가지 타입의 <br>
사용자들을 생각해 고안된 사이트입니다. <br>

첫째는 컴퓨터 부품에 대해 잘모르는 사용자들을 겨냥한 <br>
부품설명 페이지 입니다.<br>
made by 박중원 :
![그림01](https://user-images.githubusercontent.com/71188378/121768869-895ee880-cb9b-11eb-986b-bbd5200c7448.gif)

이 페이지는 컴퓨터의 각 부품을 <br>
사람의 신체 기관에 비유해 간단히 설명하고,<br>
이미지에 링크를 걸어 컴퓨터 부품에 대한 설명을 해주는 페이지 입니다.<br>
